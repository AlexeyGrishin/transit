    Define user handler for 'hello' command. Use sendBack to send data to client. It could be called any amount of times. See also Request and Response objects reference

    transit.receive 'hello', (req, res) ->
  res.sendBack "Hello #{req.user}"

    transit.receive 'echo {{params}}', (req, res) ->
  res.sendBack req.attrs.params.join(" ")
