  • ¶

    This middleware automatically calls to res.done() before calling user handler, so you do not need to care about it.

    Request extensions

    None

    Response extention

    None

    Example 

    transit = require('transit')
app = transit()

app.use transit.commandLine ack:true
  • ¶
